# FSDT Boilerplates - NextJS

Basic NextJS structure to start a new project.

## Libraries installed

### Base

- React
- NextJS

### Formatters and Code-qualiters

- Typescript
- ESLint
- Prettier

### Tests

- Jest
- Testing Library React

## Instalation

`npm install`

## Development

`npm run dev`

## Build

`npm run build`

## Tests

`npm run test`

## Start Production Server

`npm start`
