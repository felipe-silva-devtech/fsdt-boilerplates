# FSDT Boilerplates - Angular

Basic Angular structure to start a new project.

## Prerequisites

- Angular CLI
  - `npm install -g @angular/cli`

## Libraries installed

### Base

- Angular

### Formatters and Code-qualiters

- Typescript
- ESLint
- Prettier

### Tests

- Karma

## Instalation

`npm install`

## Development

`npm start`

## Build

`npm run build`

## Tests

`npm run test`

## Creating new components

`ng generate component components/[name]`
